# Ball WinForms App

Небольшое приложение на C# с использованием WinForms, в котором реализована логика движения шарика и взаимодействия с окружением.

Сделал этот проект, чтобы потренироваться в:
- работе с анимацией в WinForms
- обработке движения объектов
- базовой игровой логике
- работе с таймерами
- принципы ООП на практике
- разделение интерфейса и логики по отдельный библиотекам
- работа в команде с помощью Git и GitHub

---

## 🎯 Описание

Приложение, где шарик двигается по форме и взаимодействует с границами / объектами.

Основной фокус — на логике движения и обновлении состояния в реальном времени.

---

## 🧠 Как это устроено

- Движение реализовано через обновление координат (X, Y)
- Таймер отвечает за “тик” игры и обновление состояния
- UI (WinForms) используется для отрисовки

>  К сожалению на ранних этапах не знал как лучше сделать и в моём случае каждое движение шарика это - "нарисовать,стереть себя, переместить координату,снова нарисовать себя"
  из-за этого появляются мерцания + лаги при большой колличестве шаров на экране
